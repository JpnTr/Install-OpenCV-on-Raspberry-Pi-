# Install-OpenCV-on-Raspberry-Pi-
You can use this script for installing OpenCV on Raspberry Pi 

##How to use that?
```sh
git clone https://github.com/JpnTr/Install-OpenCV-on-Raspberry-Pi-
cd Install-OpenCV-on-Raspberry-Pi-
chmod 777 opencv_installer.sh
./opencv_installer.sh
```

Thats all :) Enjoy
